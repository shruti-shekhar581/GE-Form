# GE Form Application

A simple React application built using **React**, **Material UI (MUI)**, and **Tailwind CSS**.  
The app allows users to enter an SSO ID and perform actions like **Add User**, **Sync Password**, and **Set Ping ID** with validation and feedback using snackbars.
